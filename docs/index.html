  <div class="menu-container expand-md overlay">
    <span class="h3 logo float-left">Logo</span>

      <div class="menu-icon menu-style1 menu-icon-primary"></div>
    </button>
    <div class="menu-content menu-from-left bg-white float-right">
      <!-- this is where all the content goes -->
      <ul class="menu-list">
        <li class="menu-item"><a href="#">Link 1</a></li>
        <li class="menu-item"><a href="#">Link 2</a></li>
        <li class="menu-item"><a href="#">Link 3</a></li>
      </ul>
    </div>
  </div>
</div>

If you want to use a fixed navigation with "position-fixed", use "mt-fixed-menu" for yout content.

<div class="nav-bar bg-light position-fixed">
  <!-- your menu -->
</div>
<div class="mt-fixed-menu container">
  <!-- your content -->
</div>

Utilitities

Some examples of Utility Classes for margin, padding, border and border-radius(= round). Margin and padding can be used together with breakpoints.

<div class="mb-sm-2 py-3 border-l round">
  Vertical Padding 3 and Margin Bottom 2 for breakpoint sm and up
</div>

Grid

It has 3 Breakpoints (sm 600px, md 800px, lg 1200). Always use a ".row" and put your columns in there.

.col-md-6
.col-md-3
.col-md-3
<div class="row">
  <div class="col-md-6">
  <div class="col-md-3">
  <div class="col-md-3">
</div>
